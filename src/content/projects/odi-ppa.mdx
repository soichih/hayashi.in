The One Degree Imager — Portal, Pipeline, and Archive (**ODI-PPA**) is a web science gateway
that gives astronomers a modern interface for accessing, processing, and visualizing data from
the **WIYN Observatory's 3.5m telescope** — extending the observatory's scientific productivity
well beyond raw data acquisition.

I contributed to the portal and to browser-based astronomical visualization and analysis tools,
including **Image Explorer / ImageX** — bringing interactive image exploration of large
astronomical images into an HTML5 web application, backed by grid/cloud compute and AMQP
messaging.

- **Selected papers:** "ODI-PPA: a web-based astronomical compute archive, visualization, and
  analysis service" (*SPIE* 9152, 2014) · "ImageX: new and improved image explorer for
  astronomical images and beyond" (*SPIE*, 2016)
  [doi:10.1117/12.2233425](https://doi.org/10.1117/12.2233425) · "FRIAA: A Framework for
  Web-based Interactive Astronomy Analysis using AMQP Messaging" (ADASS XXII, 2012).
